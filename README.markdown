#Connect Four

Simple "Connect Four" implementation in Clojure.
Currently only the command line view is fully implemented.

#Build instructions

Use leiningen to build the project

    $ lein uberjar

then

    $ java -jar connectfour-1.0.0-SNAPSHOT-standalone.jar
